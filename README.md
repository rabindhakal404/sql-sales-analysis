# 📊 SQL Sales Analysis & EDA Project

## 🔍 Overview

This project performs exploratory data analysis (EDA) on a sales dataset using SQL. It focuses on extracting business insights, identifying trends, and calculating key performance metrics.

## 🎯 Objective

To analyze sales performance using SQL queries and generate meaningful insights that can support business decision-making.

## 🛠 Tools Used

* SQL (MySQL)
* Database System

## 📊 Key Analysis Performed

* Total revenue and profit analysis
* Monthly and yearly sales trends
* Top-performing products and categories
* Customer purchase behavior
* Key KPI calculations

## 📂 Project Structure

```
sql-sales-analysis/
│── sales_project.sql     # SQL queries for analysis
│── sales_dataset.csv     # Sales dataset
│── sales_clean.csv       # clean sales output
│── README.md             # Project documentation
```

## 📈 Key Insights

* Identified top revenue-generating products
* Discovered seasonal sales trends
* Calculated important business KPIs

## 🧠 What I Learned

* SQL querying (GROUP BY, ORDER BY)
* Business-focused data analysis
* Translating raw data into actionable insights

## 🚀 Outcome

A complete SQL-based analysis that transforms raw sales data into meaningful business insights.
